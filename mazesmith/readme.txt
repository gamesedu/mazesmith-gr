SETUP
=====
Upload mazesmith.js, in ASCII, to your site.  Also, upload the .gif's
to your site, in binary.

If you want to use the default form for Mazesmith, upload
mazesmith.htm, also in ASCII.

If you want to add Mazesmith to an already existent page, you will
need to do the following:

Add this to the <header> section:

<script type="text/javascript" src="mazesmith.js"></script>

Then add this anywhere you want Mazesmith to appear in your page:

<div id="mazesmith"></div>
